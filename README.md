# Google Books Explorer App

This project is geared towards applying acquired frontend development skills by building a responsive web application that consumes the Google Books API to search, browse, and save books efficiently.

- API consumption and data handling
- Modern React (Vite) project structure
- Clean UI/UX with responsive design
- State management, pagination, infinite scroll, and dark mode


# Built With

- Major languages
  - JavaScript (ES6+)

- Frameworks & Libraries
  - React (Vite)
  - Tailwind CSS

- Technologies used
  - Google Books API
  - GitHub & Git
  - Visual Studio Code

# Getting Started

This project focuses on building a real-world React application that integrates a public API and demonstrates best practices in frontend development.
It includes search functionality, pagination, infinite scroll, authentication-ready structure, favorites, and dark mode support.

# Install

- Clone the repository:
  git clone https://github.com/efoosag/google-books-app.git

- Navigate to the project directory:
  cd google-books-app

- Install dependencies:
  npm install

- Start the development server:
  npm run dev
